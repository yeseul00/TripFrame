## 1. 화면 구성 개요 (IA)

기존 상단/하단 혼재 네비게이션(v1.0)을 폐기하고, 글로벌 표준 멘탈 모델인 **하단 4-Tab 구조(v1.1)**로 통폐합한다.

* **[Tab 1] 홈 (SCR-000):** 다가오는 여행 목록, 신규 여행 생성 (e-티켓, 메일 연동)
* **[Tab 2] 일정 (SCR-001):** [v1.0 리스트 뷰 ➔ v1.1 상단 지도 + 하단 바텀시트(트리플 UI)] 기반 전체 타임라인
* **[Tab 3] 스마트 체크 (SCR-002):** [v1.0 공백/비교 탭 통합] 누락된 이동 수단 알림 + 인라인 티켓 비교 및 예매 연결
* **[Tab 4] 마이 (SCR-003):** [v1.0 유지] 설정, 계정 연동(Supabase Auth), 앱 정보

**공통 모달 (Bottom Sheets)**
* **MODAL-001 스마트 타임라인:** 항공권 등 기준 일정 클릭 시 나타나는 역산 계산 과정 뷰 (v1.0의 전체 화면 뷰에서 모달로 축소)
* **MODAL-002 일정 추가:** 분류, 시간, 내용을 입력하는 퀵 추가 모달 (v1.0 유지 및 디자인 개선)

## 2. 화면 상세 설계

### 2.1 SCR-000: 홈 (Home)
* **Hero Banner:** 화면 상단에 그라데이션 기반의 "새로운 여행 만들기" 배너 배치.
* **Smart Actions:** '메일 연동하기', 'e-티켓 스캔' 카드 버튼.
* **Trip List [v1.0 베이스]:** 사용자의 예정된 여행 카드. 여행지 이미지, 타이틀, 기간 표시.
* **Badge [v1.1 추가]:** D-Day 및 스마트 체크가 필요한 미예약 상태 뱃지 노출.

### 2.2 SCR-001: 일정 (Itinerary)
* **Map Area (Top 40%) [v1.1 추가]:** 고정된 백그라운드 지도. 경로를 잇는 Polyline과 주요 거점(호텔, 명소) 핀 마커 표시. 다이나믹 헤더 포함.
* **List Area (Bottom 60% ➔ 100%) [v1.0 고도화]:**
  * **Day Selector:** 가로 스크롤이 가능한 Pill 형태의 일자 선택기.
  * **Timeline Track:** 좌측의 굵은 세로선.
  * **Place Node [v1.1 변경]:** 기존 텍스트 박스에서 원형 아이콘 노드(Node)와 우측 카드로 변경.
  * **Transport Chip [v1.1 변경]:** 장소 간 이동 수단은 세로선 정중앙에 위치한 연결 라벨(Chip) 형태로 표시. 미예약 시 주황색(Orange), 자동 삽입 시 초록색(Green).

### 2.3 SCR-002: 스마트 체크 (Smart Check)
* **Alert Feed [v1.0 베이스]:** 누락된 이동 수단(Gap)을 리스트로 노출. (명칭 변경: 공백 감지 ➔ 스마트 체크)
* **In-place Recommendations (Expand) [v1.1 고도화]:** 카드를 탭하면 화면 이동 없이 아래로 펼쳐지며 추천 교통편, 가격, 소요 시간, 예약 오픈일, 산큐패스 비교 정보 노출. '예매하기' 버튼 제공 (기존 2 depth 화면을 인라인 아코디언으로 통합).

### 2.4 MODAL-001: 스마트 타임라인 (역산)
* **Anchor Card:** 상단에 기준이 되는 타겟 일정(예: 비행기 티켓 모양 뷰) 고정.
* **Reverse Track [v1.0 로직]:** 항공사 규정 차감, 이동 시간 차감 등을 단계별로 보여주는 마이너스(-) 타임라인 노출.
* **Result Banner [v1.1 변경]:** 화면 최하단에 시각적으로 가장 강조된 '최종 출발 권장 시간' 도출 완료 배너 제공 (v1.0 일반 텍스트에서 배너로 강조).

## 3. 인터랙션 및 트랜지션

| 트리거 | 액션 | 트랜지션 효과 |
|---|---|---|
| SCR-001 (일정) 아래로 스크롤 | 바텀 시트 전체 화면 확장 | 지도는 `scale(0.95)`, `translateY(-10px)`, `opacity(0.3)` 적용(패럴랙스). 리스트는 스피커 안전 영역까지 상승. |
| 상단 빈 영역 터치 | 바텀 시트 축소 (분할 화면) | 지도가 원래 크기와 투명도로 복귀. |
| SCR-002 (스마트 체크) 카드 탭 | 추천 교통수단 펼침 | `animate-in slide-in-from-top-2` 및 아코디언 화살표 180도 회전. |
| 타임라인 '스마트 타임라인' 뱃지 탭 | 모달 노출 | 딤(Dim) 배경 페이드인 + 바텀시트 슬라이드 업. |

## 4. 테마 및 색상 체계

기존 '에러 중심(Red)'에서 '액션 유도(Orange)'로 상태 컬러를 완화하고, 다크/라이트 모드를 동시 지원한다.

| 의미 | Light Mode | Dark Mode | 적용 대상 |
|---|---|---|---|
| **Background** | `slate-50` / `white` | `neutral-900` / `#0D0D12` | 앱 배경, 카드 배경 |
| **Primary (Brand)** | `blue-600` | `blue-400` / `blue-500` | 메인 액션, 추천 태그 |
| **USP (Reverse)** | `#7C3AED` | `#A78BFA` | 스마트 타임라인(역산) 강조 영역 |
| **Warning (Gap)** | `orange-500` | `orange-400` | 미예약 구간, 확인 필요 알림 (v1.0 Red에서 대체) |
| **Success (Auto)** | `emerald-500` | `emerald-400` | 시스템 자동 삽입 구간 |

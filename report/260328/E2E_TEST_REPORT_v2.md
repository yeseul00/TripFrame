## 결론

신규 디바이스(Windows 11)에서 E2E 테스트 **33/33 100% 통과** 달성.

**핵심 교훈**:
1. Expo React Native Web에서는 `getByRole('button')` 대신 `locator('text=')` 사용
2. 동일 텍스트 중복 시 `.first()` / `.last()` / `{ exact: true }` 로 strict mode 해소
3. App.tsx의 DOM 렌더 순서(화면 → 탭바)를 고려한 `.last()` 활용
4. Playwright `webServer` 서버 불안정 문제는 셀렉터 수정으로 연쇄 타임아웃 해소

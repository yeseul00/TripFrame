## 🐛 발견된 이슈

### Issue #1: 온보딩 웹 렌더링 레이아웃 깨짐 (우선순위: 중간)
- **설명**: `OnboardingScreen.tsx`의 FlatList horizontal 슬라이드가 웹 환경에서 레이아웃 미적용. 슬라이드 제목이 노출되지 않고, 배경색이 흰색으로 렌더링됨.
- **영향**: 웹 기반 E2E 테스트에서 온보딩 화면 시각 검증 불가. 네이티브(iOS/Android) 동작에는 영향 없음.
- **원인 추정**: NativeWind `flex-1` + `w-screen` 클래스가 웹에서 올바르게 적용되지 않음. FlatList의 `horizontal` prop이 웹에서 완전히 지원되지 않을 수 있음.
- **권장 해결**:
  1. 온보딩 슬라이드 컨테이너에 `style={{ backgroundColor: '#0F0F13' }}` 인라인 적용
  2. FlatList 대신 ScrollView + 수동 인덱스 관리로 교체 (웹 호환성↑)
  3. 또는 웹 전용 CSS 폴백 추가
- **Phase 5 대응**: 네이티브 전용 테스트(Maestro) 전환 시 해소 예정

## P1. 짐 많은 여행자 — "김수영 (35세, 가족여행)"

**배경**: 4인 가족 여행. 큰 캐리어 4개. 대중교통은 체력 소모가 크다.

**선호도 설정**:
```
luggageSize: LARGE
transportPreference: TAXI
timeBuffer: RELAXED
```

**기대 동작**:
- 이동 수단 추천 → 택시 / 렌터카 1순위
- 역산 시 수하물 찾기 시간(+20분) 자동 반영
- 예약 버튼이 명확하게 노출됨

**검증 시나리오**:
- 하카타 → 유후인 구간에서 택시가 최상위 추천인지 확인
- 인원 4명 설정 시 합산 요금이 1인 × 4로 정확히 계산되는지 확인

## P2. 알뜰 배낭여행자 — "박지민 (27세, 솔로여행)"

**배경**: 혼자 여행, 기내 캐리어 1개. 비용 최소화가 최우선.

**선호도 설정**:
```
luggageSize: CARRY_ON
transportPreference: PUBLIC
timeBuffer: RELAXED
```

**기대 동작**:
- 이동 수단 추천 → 버스 / 열차 1순위
- 저렴한 옵션이 목록 상단에 위치
- 예약 불필요 옵션 우선 표시

**검증 시나리오**:
- 하카타 → 유후인 구간에서 고속버스 또는 열차가 1순위인지 확인
- 1인 요금이 가장 낮은 옵션이 상위에 노출되는지 확인

## P3. 빡빡한 일정의 비즈니스 여행자 — "이준혁 (42세, 출장여행)"

**배경**: 미팅 시각이 고정되어 있어 이동 시간을 최소화해야 함.

**선호도 설정**:
```
luggageSize: CARRY_ON
transportPreference: ANY
timeBuffer: TIGHT
```

**기대 동작**:
- 역산 결과가 TIGHT 기준으로 최소 버퍼만 포함
- 소요시간이 짧은 이동 수단이 우선 추천
- Gap 감지 시 위험도를 명확히 강조

**검증 시나리오**:
- 역산 화면에서 TIGHT 설정 시 여유 시간이 줄어드는지 확인
- 소요시간 90분 이내 옵션이 1순위로 추천되는지 확인
